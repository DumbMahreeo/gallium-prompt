# Accepted colors

Accepeted effect arguments are:

| Effect name        | Alias       |
| ------------------ | ----------- |
| `bold`             | `bright`    |
| `faint`            |             |
| `italic`           |             |
| `underline`        |             |
| `double_underline` |             |
| `slow_blink`       |             |
| `fast_blink`       |             |
| `reverse`          | `inverse`   |
| `erase`            | `invisible` |
| `striketrough`     |             |

## Note

Some of those (such as blink and double underline) might not work on your terminal.
